## 🚀 Quick Start

#### Windows (Recommended)

1. Extract the ZIP anywhere.

2. Double-click:

```text
START_ENTCare_AI.bat
```

3. Browser opens automatically.

4. Access:

```text
http://localhost:5100
```

5. Begin exploring ENT diseases, assessments, emergency guidance, and educational resources.

## 🔧 Troubleshooting

#### Application Does Not Open

Open:

```text
START_ENTCare_AI.bat
```

or browse to:

```text
http://localhost:5100
```

using Chrome, Microsoft Edge, or Firefox.

#### AI Providers Do Not Respond

Review:

- API configuration
- Internet connection
- Selected provider settings

#### Assessment Appears Incorrect

Review:

- Symptoms entered
- Duration
- Side affected
- Severity
- Associated symptoms

Remember that educational assessments **DO NOT** replace professional ENT consultation.

#### Report Issues

Refresh the browser and regenerate the report.
